# Code repository to reproduce analyses from Giotti et al 'Single cell view of pleural mesothelioma reveals substantial differences in the tumor microenvironment associated with a cancer-intrinsic, sarcomatoid cell signature'

Scripts are organized to group together figures generated from the same Seurat object. To download Seurat objects for each compartment analyzed, inclusive of all the metadata, please download them from GEO GSE190597.

# R packages required are listed in the script load_libraries.R and custom functions used throughout can be sourced from useful_functions.R
